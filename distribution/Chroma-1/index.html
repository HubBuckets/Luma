Chroma

A library by Neil Panchal for the Processing programming environment.
Last update, 03/28/2015.

Chroma is color conversion library that implements perceptually linear color spaces such as CIELab & CIE-LCH.
Chroma objects can be built using standard color spaces such as RGB, HCL, HCV, or human friendly color spaces such as CIE-Lab & CIE-LCH; The latter being perceptually linear. It is inspired by the popular Chroma.js JavaScript library by Gregor Aisch for working with colors in visualizations (https://github.com/gka/chroma.js).

Download

Download Chroma version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted Chroma folder into the libraries folder of your Processing sketches. Reference and examples are included in the Chroma folder.
